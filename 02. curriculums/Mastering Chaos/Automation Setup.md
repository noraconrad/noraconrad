## Step 1: Email Scheduling Setup

### Gmail Scheduling Features

**Built-in Gmail Scheduling:**

1. Compose your email normally using your templates
2. Click the arrow next to "Send" (don't click Send directly)
3. Select "Schedule send"
4. Choose timing:
   - Tomorrow morning at 8 AM
   - Monday at 9 AM
   - Pick date & time (custom)
5. Gmail holds the email and sends it at your specified time

### Outlook Scheduling Features

**Outlook Delay Delivery:**

1. Compose your email
2. Click "Options" tab in the ribbon
3. Select "Delay Delivery"
4. Check "Do not deliver before" and set date/time
5. Click "Close" and send normally
6. Email sits in Outbox until scheduled time

**Outlook Rules for Automatic Scheduling:**

1. File â†’ Rules and Alerts â†’ New Rule
2. "Apply rule on messages I send"
3. Add conditions (e.g., contains "proposal" in subject)
4. Set action: "delay delivery by X hours"
5. Automatically schedules certain email types

### Apple Mail Scheduling Options

**Using Send Later (Third-party):**

- **Send Later App** ($5): Integrates with Apple Mail
- **Mail Pilot** ($10): Full email client with scheduling
- **Airmail** ($3/month): Professional email client with scheduling

**Manual Scheduling Workaround:**

1. Save email as draft
2. Set calendar reminder for when to send
3. Send when reminder triggers
4. Less elegant but works with built-in tools

### Advanced Scheduling Tools

**Boomerang** (Gmail/Outlook)
- Free plan: 10 scheduled emails per month
- Pro plan: $5/month for unlimited scheduling
- Features: Schedule send, remind if no reply, pause inbox

**Mixmax** (Gmail)
- Free plan: Basic scheduling
- Premium: $9/month with advanced features
- Features: Schedule send, email tracking, templates with scheduling

**Right Inbox** (Gmail)
- Free plan: Limited scheduling
- Premium: $7.95/month
- Features: Recurring emails, send later, email reminders

## Step 3: Follow-Up Automation

### Proposal Follow-Up Sequences

You send a proposal and then... wait. Or manually remember to follow up in a few days. But we can automate this with CRM tools like Honeybook, Dubsado, or apps mentioned below.

**The Automated Proposal Follow-Up Sequence:**

```
DAY 0: Proposal sent
"Hi [Name], attached is your custom proposal for [project]. Questions? Just reply here!"

DAY 3: First follow-up (if no response)
"Hi [Name], wanted to check if you had any questions about the [project] proposal I sent Tuesday. Happy to hop on a quick call to discuss any details. What questions can I answer?"

DAY 7: Second follow-up (if no response)
"Hi [Name], following up once more on the [project] proposal. I know decisions take time! If you need more time to review, just let me know. If you'd like to move forward, simply reply 'approved' and I'll send the contract."

DAY 14: Final follow-up (if no response)
"Hi [Name], I'll assume the [project] isn't a priority right now, which is completely understandable. Feel free to reach out when you're ready to move forwardâ€”this proposal will remain valid for the next 30 days."
```

### Tools for Follow-Up Automation

**Boomerang** (Gmail/Outlook)
- "Respond If No Reply" feature
- Set up automatic follow-ups if client doesn't respond
- Customizable timing and message content

**Mixmax** (Gmail)
- Email sequences for automated follow-ups
- Conditional logic based on recipient actions
- Integration with CRM systems

**Manual Follow-Up System:** If automation tools aren't in budget:

1. Create calendar reminders for follow-up dates
2. Use task management system to track proposal status
3. Create follow-up email templates for consistency
4. Set weekly review to check proposal statuses

## Step 4: Calendar Integration & Meeting Automation

### Automated Meeting Coordination

Gone are the days of:

> "How about Tuesday?" "Tuesday doesn't work, what about Wednesday?" "Wednesday afternoon?" "Morning is better"...

Scheduling links handle coordination automatically.

### Setting Up Scheduling Automation

**Calendly Setup:**

1. Sign up at calendly.com (free plan available)
2. Connect your calendar (Google, Outlook, iCloud)
3. Set your availability:
   - Business hours
   - Buffer time between meetings
   - Meeting types (consultation, project check-in, etc.)
4. Create event types:
   - 15-min quick call
   - 30-min consultation
   - 60-min strategy session
5. Customize booking page:
   - Your photo and bio
   - Meeting descriptions
   - Required information from bookers
6. Set up notifications:
   - Email confirmations
   - Reminder emails
   - Calendar invites with video call links

**Google Calendar Appointment Slots:**

1. Open Google Calendar
2. Click "+" to create event
3. Select "Appointment slots"
4. Set recurring availability
5. Share booking link with clients
6. Free alternative to paid scheduling tools

**Acuity Scheduling:**

- More customization than Calendly
- Payment integration for paid consultations
- Intake forms and questionnaires
- Team scheduling if you have employees
