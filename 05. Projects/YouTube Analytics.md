## Description

Personal business metrics tracking for @NoraConrad YouTube channel

## Details

A custom dashboard pulling YouTube API data to track channel growth, revenue, audience behavior, and content performance over a decade of publishing.

## Technologies

- YouTube API
- Google Sheets
- Google Apps Script
- Custom Dashboards

## Features

- Weekly subscriber growth tracking
- Revenue breakdown by source (YouTube, sponsorships, products)
- Content performance by video type and topic
- Audience demographics and retention
- Year-over-year growth comparisons

## Challenges

Consistent data collection and API reliability; isolating revenue sources; identifying content patterns that drive growth.

## Outcomes

Clear visibility into channel health and revenue drivers; framework used to inform content strategy and product decisions.
