# Warehouse Management System

**Status:** Building  
**External Site:** [macpac.space](https://www.macpac.space)

## Description

Multi-client inventory and warehouse management system

## Details

A web-based WMS built to streamline inventory operations for CPG fulfillment. Designed after 10 years working in warehouse operations at MacPac Fulfillment, managing Hiball Energy inventory and order fulfillment. The system handles SKU tracking, location mapping, order processing, and inventory reconciliation.

## Technologies

- Vercel
- Supabase
- React
- PostgreSQL

## Features

- Real-time inventory tracking by location and SKU
- Order processing and fulfillment workflows
- Inventory movement logging and reconciliation
- SKU master data management
- Reporting on stock levels and movement velocity

## Challenges

Integrating with existing warehouse processes; designing for non-technical warehouse staff; real-time accuracy across high-volume SKU movement.

## Outcomes

Operational tool reducing manual tracking and improving order accuracy; demonstrates deep domain knowledge in CPG fulfillment operations.
