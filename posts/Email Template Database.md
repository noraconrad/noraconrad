## Building Your Template Library

### Action Steps:

1. **[ ] Audit Your Current Emails**
   - Go through your sent folder
   - Identify patterns (what emails do you send repeatedly?)
   - Group by type/purpose

2. **[ ] Create Template for Each Pattern**
   - Use the template structure provided above
   - Include customization notes
   - Test with real scenarios

3. **[ ] Organize in Central Location**
   - Notion, Google Drive, or email client
   - Make searchable by type/situation
   - Link to related processes

4. **[ ] Test with Real Emails**
   - Use templates for actual client communications
   - Note what needs adjustment
   - Refine based on results

5. **[ ] Set Up Template System**
   - Gmail Canned Responses, Apple Mail Signatures, Superhuman templates
   - Create keyboard shortcuts for fast access
   - Train yourself to use templates consistently
